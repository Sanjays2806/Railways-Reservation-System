# Railway Reservation System

Railway Reservation System is a simple web-based application developed using Python, HTML, CSS, JavaScript, and MongoDB.

## Project Description

This application allows users to search for trains, enter passenger details, and reserve railway tickets through a web interface. It demonstrates basic reservation system functionality with frontend and backend integration.

## Technologies Used

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python

Database:
- MongoDB

## Features

- Train search functionality
- Ticket booking system
- Passenger information management
- Form validation using JavaScript
- Data storage using MongoDB
